Création de logo

Pour ce projet de PAO, nous avions le droit de créer un logo sur le sujet que nous voulons. J'ai donc choisi de créer un logo pour le logiciel de création musicale d'Apple "Logic Pro". Actuellement, nous en somme à la version 10 du logiciel, j'ai donc naturellement choisi de faire le logo pour la 11ème version du logiciel qui est bien évidemment ficitve pour l'instant. Dans ce logo, on distingue les bandes grises qui représente les égaliseurs présent dans le logiciel.
