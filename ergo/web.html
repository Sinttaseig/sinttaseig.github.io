Space-Golf

Version desktop

L'image ci-dessus est une maquette de site web représentant l'ouverture de plusieurs parcours de golf sur la Lune. Il fallait donc réaliser une interface en rapport avec le thème mais aussi au gout du jour. A l'heure où le flat design est bien intégré dans notre quotidien, ce site correspond à ce mouvement actuel. Le responsive design etait bien evidemment un règle importante à la conception de cette maquette. Nous avons donc ici la version Desktop.

Version smartphone

De plus en plus de personne utilisent leur smartphone pour consulter des pages web. C'est pour ceci qu'une version Smartphone de la maquette a été réalisé. Cette version devait être la plus homogène possible avec la version Desktop afin que l'utilisateur ne soit pas perdu entre la version Desktop et Smartphone. Une ergonomie simple et intuitive à donc été choisit pour ce projet.
